# Multi-Agent Blog Demo

Spring Boot + Vue 3 blog system for testing multi-AI-agent collaboration.

## Architecture
- `backend/` - Spring Boot 3.x + MyBatis-Plus + MySQL
- `frontend/` - Vue 3 + Element Plus + Vite

## Agents
- **Backend Agent** - API development, database, security
- **Frontend Agent** - Vue components, pages, API calls
